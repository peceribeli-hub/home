Prompt de Clareza
e Execução

Como transformar reuniões, transcrições e textos longos em ações claras, passo a passo — sem retrabalho.

Contexto

Para que serve este prompt

Toda vez que você tiver um conteúdo longo para processar — transcrição de reunião, notas brutas, briefing de cliente — este prompt garante que a IA trabalhe na ordem certa antes de criar qualquer arquivo.

O problema mais comum: a IA vai direto para a execução sem entender como as peças se encaixam. Isso gera retrabalho e resultados genéricos.

Com este prompt, a IA primeiro mapeia a lógica do conteúdo, organiza tudo em tabelas, confirma com você e só então executa — com instruções que qualquer pessoa consegue seguir.

Fluxo

4 etapas — sempre nesta ordem

A IA nunca pula etapas. Você não precisa fazer nada até a Etapa 2.

0
MAPA DE CLAREZA
Antes de qualquer tarefa, a IA mostra como as peças se encaixam: quais são os componentes, como um alimenta o outro e qual é a hierarquia.
1
EXTRAÇÃO
Organiza todo o conteúdo em tabelas: decisões tomadas, tarefas (quem / o quê / quando), números e metas, pessoas e papéis, ambiguidades.
2
CONFIRMAÇÃO
A IA faz no máximo 5 perguntas. Você valida o mapa e responde as dúvidas. Só depois de confirmado qualquer arquivo é criado.
3
EXECUÇÃO
A IA executa tudo de uma vez: cria os arquivos e entrega as instruções com passo a passo clique por clique, sem novas perguntas no meio.
Regra importante: a IA nunca pula etapas. Se você não confirmar na Etapa 2, ela não avança para a Etapa 3.

Uso

Como usar o prompt

Siga os passos abaixo toda vez que precisar processar um conteúdo longo.

1
Abra o Cowork (Claude) em uma nova aba.
2
Vá até a seção "O Prompt Completo" nesta página e clique em Copiar Prompt.
3
Cole no campo de mensagem do Cowork.
4
Substitua [TIPO DE ENTRADA] pelo tipo do seu conteúdo — por exemplo: transcrição de reunião.
5
Substitua [OBJETIVO FINAL] pelo que você quer como saída — por exemplo: redistribuir tarefas no ClickUp.
6
Cole o seu conteúdo no final, após a linha [COLE SEU CONTEÚDO AQUI].
7
Envie e aguarde. A IA vai mostrar o Mapa de Clareza primeiro. Leia com atenção.
8
Se algo no mapa estiver errado, corrija antes de confirmar.
9
Responda as perguntas da Etapa 2.
10
Aguarde a execução completa com os arquivos entregues e o passo a passo.

Prompt completo 
MODO: Mapa de Clareza → Análise → Confirmação → Execução

Tipo de entrada: [transcrição de reunião / notas brutas / documento / texto longo]
Objetivo final: [ex: redistribuir tarefas / criar KPIs / montar plano de ação / outro]

---

ETAPA 0 — MAPA DE CLAREZA (antes de qualquer tarefa ou arquivo)

Leia tudo. Antes de listar tarefas, me mostre como as peças se encaixam.

Produza um mapa com este formato:

COMPONENTES IDENTIFICADOS
Liste cada peça do sistema mencionada no conteúdo
(funis, processos, papéis, etapas, documentos).

COMO SE CONECTAM
Para cada conexão, escreva uma linha:
→ [Peça A] alimenta [Peça B] porque [motivo simples]
→ [Peça B] depende de [Peça A] para funcionar

HIERARQUIA
Monte uma estrutura visual em texto mostrando o que vem antes e o que vem depois:

[Nível 1 — base / origem]
  └── [Nível 2 — alimentado pelo nível 1]
        └── [Nível 3 — resultado final]

ONDE A LÓGICA AINDA NÃO ESTÁ CLARA
Liste qualquer ponto onde a conexão entre peças ficou vaga ou contraditória.

Regra: use linguagem simples. Escreva como se fosse explicar para alguém
que nunca viu este projeto.

---

ETAPA 1 — EXTRAÇÃO (não crie nenhum arquivo ainda)

Após o mapa, extraia e apresente como tabelas estruturadas:

| Categoria       | O que extrair                                   |
|-----------------|-------------------------------------------------|
| Decisões        | O que foi decidido, por quem                    |
| Tarefas         | Quem | O quê | Até quando | Prioridade          |
| Números / metas | Valores, datas, percentuais mencionados         |
| Pessoas         | Nomes, papéis, responsabilidades citadas        |
| Contexto-chave  | Problemas identificados, riscos, oportunidades  |
| Ambiguidades    | O que ficou vago, contraditório ou incompleto   |

Seja direto. Sem parágrafos explicativos — só tabelas e listas.

---

ETAPA 2 — CONFIRMAÇÃO

Após as tabelas, faça no máximo 5 perguntas. Apenas sobre:
- Pontos do mapa de clareza que precisam de validação
- Ambiguidades que bloqueiam a execução
- Decisões que dependem da minha preferência (não assuma)

Formato das perguntas: numeradas, uma linha cada, sem contexto longo.

Encerre com:
"Confirme o mapa e responda as perguntas. Só depois vou executar."

---

ETAPA 3 — EXECUÇÃO

Aguardar confirmação. Após minha resposta:
- Atualize o mapa se necessário
- Execute tudo de uma vez
- Sem novas perguntas durante a execução
- Para cada ação, escreva:
  ✅ O que fazer (uma frase)
  📍 Onde fazer (ex: Abra o ClickUp → clique em Tarefas → selecione o projeto X)
  👆 Passo a passo numerado com cliques exatos
  ✔️ Como saber que deu certo (o que vai aparecer na tela)
- Entregue os arquivos ao final
