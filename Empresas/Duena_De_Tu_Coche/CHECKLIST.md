# 🚀 ROADMAP: Lançamento Dueña de tu Coche

Este é o plano de ação para transformar este infoproduto em uma máquina de vendas.

---

## 📋 Checklist de Execução

### 1. Produto (E-book) — O Coração do Funil
- [ ] **Expansão do Ghostwriting:** Transformar as 15 regras básicas em capítulos detalhados e práticos.
- [ ] **Revisão Cultural:** Ajustar o tom do espanhol para ser empoderador, técnico e empático.
- [ ] **Layout Final:** Formatação estruturada em Markdown para fácil conversão em PDF/EPUB.

### 2. Marketing & Copy (Vendas)
- [ ] **Headline de Impacto:** A promessa irresistível que para o scroll.
- [ ] **Sales Copy (Long-form):** Texto persuasivo focado em dor (golpes de mecânicos) e prazer (independência).
- [ ] **Quebra de Objeções:** Manual de respostas para "Eu não entendo de carros", "É caro", etc.
- [ ] **Mockups 3D:** Imagens premium do e-book em dispositivos (Tablet/Celular) para a página.

### 3. Landing Page (A Vitrine)
- [ ] **Design Ultra Premium:** Estrutura HTML/CSS com Glassmorphism, Dark Mode e acentos Magenta.
- [ ] **Otimização Mobile-First:** 90% do tráfego virá de anúncios no celular.
- [ ] **Integração de Tracking:** Configurar Pixel do Meta e API de Conversão.

### 4. Checkout & Tráfego (O Dinheiro)
- [ ] **Conexão Onebula/WhatsApp:** Configuração do link que converte em conversa.
- [ ] **Criativos Meta Ads:** 3 variações de anúncios (Estático, Vídeo, Carrossel).
- [ ] **Lançamento do Teste:** Subir a campanha ABO 48h conforme diretriz de Tráfego Perpétuo.
